# APIs

Different architectural styles for designing APIs.

## Styles

- [[REST]] — Resource-based HTTP APIs
- [[GraphQL]] — Typed query language for APIs
- [[gRPC]] — High-performance RPC using Protocol Buffers
- [[SOAP]] — XML-based protocol (legacy)
- [[WebSocket]] — Full-duplex communication

## Real-Time Communication

- [[Socket.io]] — Real-time event-based communication library

## Tools

- **cURL** — Command-line tool for HTTP requests
- **Postman** — API testing and development
- **Insomnia** — REST client
