```bash
Delete a word
Keyboard Shortcut: ⌥←Delete
Send Hex Code - 0x17
```

```bash
Delete a line
Keyboard Shortcut: ⌥↑←Delete
Send Hex Code - 0x15
```

```bash
Jump Left
Keyboard Shortcut: ⌥←
Action: Send Escape Sequence
Esc+: b
```

```bash
Jump Right
Keyboard Shortcut: ⌥→
Action: Send Escape Sequence
Esc+: f
```
