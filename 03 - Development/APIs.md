Tools and standards for API development and testing.

## Tools

```shell
# REST client
brew install httpie

# API design and testing
brew install postman
```

## Concepts

- **REST** — Resource-based HTTP APIs
- **GraphQL** — Query language for APIs; clients request exactly the data they need
- **gRPC** — High-performance RPC framework using Protocol Buffers
- **OpenAPI** — Standard specification format for REST APIs
