# Front-End Frameworks

Popular frameworks and libraries for building user interfaces.

## Frameworks

- **jQuery** — DOM manipulation library
- [[React]] — Component-based UI library
- [[Next.js]] — React framework for production
- [[Angular]] — Full MVC framework
  - See: [Map vs MergeMap vs SwitchMap](https://dev.to/bhagatparwinder/map-vs-mergemap-vs-switchmap-5gee)
- [[Vue]] — Progressive JavaScript framework
- [[Nuxt.js]] — Vue.js meta-framework
- [[Svelte]] — Compiler-based framework
- [[SvelteKit]] — Svelte meta-framework
- [[Astro]] — Static site generator with islands
