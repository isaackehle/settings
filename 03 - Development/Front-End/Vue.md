# <img src="https://github.com/vuejs.png" width="24" style="vertical-align: middle; border-radius: 4px;" /> Vue.js

A progressive JavaScript framework for building user interfaces.

## Installation

```shell
npm create vue@latest
```

## Usage

```vue
<template>
  <div>
    <h1>{{ message }}</h1>
  </div>
</template>

<script>
export default {
  data() {
    return {
      message: 'Hello, Vue!'
    }
  }
}
</script>
```
