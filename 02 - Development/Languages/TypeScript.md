# <img src="https://github.com/microsoft.png" width="24" style="vertical-align: middle; border-radius: 4px;" /> TypeScript

Typed superset of JavaScript for safer large-scale frontend and backend projects.

## Installation

```shell
brew install node
npm install --global typescript
```

## Configuration

```shell
tsc --init
```

## Start / Usage

```shell
tsc --version
```
