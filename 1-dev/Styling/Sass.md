# <img src="https://github.com/sass.png" width="24" style="vertical-align: middle; border-radius: 4px;" /> Sass

Sass is a preprocessor scripting language that is interpreted or compiled into CSS.

## Installation

```shell
npm install -g sass
```

## Usage

Compile Sass to CSS:

```shell
sass input.scss output.css
```

Watch for changes:

```shell
sass --watch input.scss output.css
```

## Features

- Variables
- Nesting
- Mixins
- Functions
- Partials
