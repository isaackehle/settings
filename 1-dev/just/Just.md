# <img src="https://github.com/casey.png" width="24" style="vertical-align: middle; border-radius: 4px;" /> Just

A command runner (simpler alternative to Make).

## Installation

```shell
brew install just
```

## Configuration

No basic configuration required.

## Usage

Create a `justfile` in your project root with recipes, then run:

```shell
just <recipe-name>
```
