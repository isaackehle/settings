# Settings

Personal macOS development environment setup guide, organized as an [Obsidian](https://obsidian.md) vault.

## Opening in Obsidian

Open the root of this repository as a vault in Obsidian. Start at [[Home]] for the full index.

## MacOS Computer Name

```shell
sudo scutil --set HostName MyComputerName
sudo scutil --set ComputerName MyComputerName
sudo scutil --set LocalHostName MyComputerName
```
