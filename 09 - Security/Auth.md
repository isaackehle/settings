# Authentication & Passwords

Password managers and MFA tools.

## Installation

```shell
# 1Password (CLI + app)
brew install --cask 1password
brew install 1password-cli

# Authy — TOTP MFA
brew install --cask authy

# KeePassX — open-source password manager
brew install --cask keepassx
```
