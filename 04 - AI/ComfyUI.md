# <img src="https://github.com/Comfy-Org.png" width="24" style="vertical-align: middle; border-radius: 4px;" /> ComfyUI

Powerful and modular node-based GUI for Stable Diffusion. More flexible than Automatic1111 for building custom pipelines.

## Installation

```shell
git clone https://github.com/Comfy-Org/ComfyUI.git
cd ComfyUI
pip install -r requirements.txt
```

## Start / Usage

```shell
cd ComfyUI && python main.py
```
