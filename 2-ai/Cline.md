# <img src="https://github.com/cline/cline.png" width="24" style="vertical-align: middle; border-radius: 4px;" /> Cline

Autonomous AI agent for VS Code and terminal that can read/write files, run commands, and browse the web.

## Installation

### VS Code Extension
Search for "Cline" in the VS Code Extensions Marketplace or install via CLI:
```shell
code --install-extension cline.cline
```

### CLI Tool
```shell
npm install -g @cline/cli
```

## Configuration

No basic configuration required. Setup is handled within the VS Code extension settings panel (API keys, provider selection, and custom instructions).

## Start / Usage

### VS Code Extension
Start: Open the Cline icon from the VS Code Activity Bar.

### CLI Tool
```shell
# Run the Cline agent in your terminal
cline
```
