# Rust

Systems programming language focused on memory safety and performance.

## Installation

```shell
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

This installs `rustup`, `rustc`, and `cargo`.

## Usage

```shell
# Update Rust
rustup update

# Create a new project
cargo new my-project

# Build and run
cargo build
cargo run

# Run tests
cargo test
```
