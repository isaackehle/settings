# <img src="https://github.com/obsproject.png" width="24" style="vertical-align: middle; border-radius: 4px;" /> Multimedia

Audio, video, graphics, and image tools.

## Graphics & Image Editing

```shell
brew install --cask gimp
brew install --cask inkscape
brew install imagemagick
brew install --cask omnigraffle
```

## Color Tools

```shell
brew install --cask colorsnapper
```

## Video

```shell
brew install ffmpeg
brew install --cask handbrake
brew install --cask vlc
brew install --cask kodi
```

## Audio & Music

```shell
brew install --cask spotify
brew install --cask amazon-music
```

ID3 tag editing:

```shell
brew install id3-editor
brew install --cask mp3tag
```

## Video Encoding

```shell
brew install x264
brew install webp
```

## Terminal Colouriser

[grc](http://kassiopeia.juls.savba.sk/~garabik/software/grc/) adds colour to common command output:

```shell
brew install grc
```

## Configuration

No basic configuration required.

## Start / Usage

- GUI apps: Open the app from Applications.

```shell
ffmpeg -version
```
