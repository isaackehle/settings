# <img src="https://github.com/HandBrake.png" width="24" style="vertical-align: middle; border-radius: 4px;" /> DVDs & Disc Burning

Tools for ripping and burning optical media.

## Installation

```shell
# DVD/Blu-ray ripping and conversion
brew install --cask handbrake

# Disc burning
brew install --cask burn

# Image processing for disc covers / artwork
brew install imagemagick
```

## Configuration

No basic configuration required.

## Start / Usage

Start: Open the app from Applications.
