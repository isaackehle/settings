# Chats & Communication

Messaging and social apps.

## Installation

```shell
brew install --cask slack
brew install --cask microsoft-teams
brew install --cask telegram
brew install --cask skype
brew install --cask discord
```

### Email

```shell
brew install --cask kiwi-for-gmail
```
