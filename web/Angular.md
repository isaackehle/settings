# <img src="https://github.com/angular.png" width="24" style="vertical-align: middle; border-radius: 4px;" /> Angular

Angular CLI for building Angular applications.

## Installation

```shell
npm install -g @angular/cli
```

## Configuration

No basic configuration required.

## Usage

```shell
ng new my-app
cd my-app
ng serve
```
