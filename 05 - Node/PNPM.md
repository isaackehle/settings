# PNPM

Fast, disk-efficient package manager. Uses a content-addressable store so packages are never duplicated across projects.

## Installation

```shell
brew install pnpm
```

Or via Corepack (included with Node.js):

```shell
corepack enable pnpm
```

## Usage

```shell
# Install dependencies
pnpm install

# Add a package
pnpm add lodash

# Run a script
pnpm run build
```
