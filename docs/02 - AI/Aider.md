# <img src="https://github.com/Aider-AI.png" width="24" style="vertical-align: middle; border-radius: 4px;" /> Aider

AI pair programming tool in your terminal. Works with local models and commercial APIs, and directly commits changes to your git repo.

## Installation

```shell
pip install aider-chat
```

## Start / Usage

```shell
aider
```
