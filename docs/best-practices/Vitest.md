# <img src="https://github.com/vitest-dev.png" width="24" style="vertical-align: middle; border-radius: 4px;" /> Vitest

A blazing fast unit test framework powered by Vite.

## Installation

```shell
npm install -D vitest
```

## Configuration

Add to `package.json`:

```json
{
  "scripts": {
    "test": "vitest"
  }
}
```

## Usage

```shell
npm test
# or
npx vitest
```
