# <img src="https://github.com/parcel-bundler.png" width="24" style="vertical-align: middle; border-radius: 4px;" /> Parcel

Zero-configuration build tool that works out of the box.

## Installation

```shell
npm install --save-dev parcel
```

## Configuration

No basic configuration required. Parcel auto-detects entry points and transforms.

## Start / Usage

```shell
npx parcel index.html        # Dev server
npx parcel build index.html  # Production build
```

Key features: zero config, supports JS/TS/CSS/HTML automatically, code splitting, Babel and PostCSS built in.
