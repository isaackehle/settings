# <img src="https://github.com/grafana.png" width="24" style="vertical-align: middle; border-radius: 4px;" /> Load Testing

Tools for testing application performance under load.

## k6

Scriptable load testing tool with a JavaScript API.

```shell
brew install k6
```

Run a test:

```shell
k6 run script.js
```

## oha

Fast HTTP load generator (Rust-based) for quick API benchmarking.

```shell
brew install oha
```

## Configuration

No basic configuration required.
