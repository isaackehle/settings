# <img src="https://github.com/oven-sh.png" width="24" style="vertical-align: middle; border-radius: 4px;" /> Bun

A fast JavaScript runtime and package manager that aims to be a drop-in replacement for Node.js.

## Installation

```shell
brew tap oven-sh/bun
brew install bun
```

Or via the official installer script:

```shell
curl -fsSL https://bun.com/install | bash
```

## Configuration

No basic configuration required.

## Usage

```shell
# Install dependencies
bun install

# Add a package
bun add lodash

# Run a script
bun run build
```
