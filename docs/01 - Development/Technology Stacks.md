# Technology Stacks

Common technology stacks for web development.

## Popular Stacks

- **LAMP** — Linux, Apache, MySQL, PHP
- **MEAN** — MongoDB, Express.js, Angular, Node.js
- **MERN** — MongoDB, Express.js, React, Node.js
- **Ruby on Rails** — Ruby, Rails framework, PostgreSQL
- **Django** — Python, Django framework, PostgreSQL
- **Spring Boot** — Java, Spring framework, various databases
