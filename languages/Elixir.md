# <img src="https://github.com/elixir-lang.png" width="24" style="vertical-align: middle; border-radius: 4px;" /> Elixir

Functional language on the Erlang VM for fault-tolerant distributed systems.

## Installation

```shell
brew install elixir
```

## Configuration

```shell
mix local.hex --force
```

## Start / Usage

```shell
iex
```
