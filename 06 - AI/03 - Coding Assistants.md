# AI Coding Assistants

Tools to integrate AI directly into your development workflow.

## Terminal Agents

| Tool | Description |
| --- | --- |
| [[Claude Code]] | Anthropic's official CLI coding agent |
| [[OpenCode]] | Terminal TUI agent, supports local and cloud models |
| [[Crush]] | Charm's terminal TUI agent with LSP and MCP support |
| [[Google Gemini CLI]] | Google's terminal agent with web search and MCP |
| [[Aider]] | Pair programmer in your terminal, commits to git |
| [[Open Interpreter]] | Run code (Python, JS, Shell) via LLM locally |
| [[OpenHands]] | Autonomous AI dev platform (formerly OpenDevin) |
| [[OpenShell]] | NVIDIA's sandboxed runtime for running agents safely |
