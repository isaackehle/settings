# <img src="https://github.com/OpenInterpreter.png" width="24" style="vertical-align: middle; border-radius: 4px;" /> Open Interpreter

Lets LLMs run code (Python, Javascript, Shell, and more) locally on your computer. Chat with it in your terminal to control your computer.

## Installation

```shell
pip install open-interpreter
```

## Start / Usage

```shell
interpreter
```
