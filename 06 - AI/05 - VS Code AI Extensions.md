# VS Code AI Extensions

VS Code extensions that bring AI coding assistance directly into the editor.

| Extension | ID | Description |
| --- | --- | --- |
| [[GitHub Copilot]] | `GitHub.copilot` | AI pair programmer; supports Ollama local models |
| [[Continue]] | `Continue.continue` | Chat, inline edit, autocomplete with local/cloud models |
| [[Cline]] | `saoudrizwan.claude-dev` | Autonomous agent with file/terminal/browser access |
