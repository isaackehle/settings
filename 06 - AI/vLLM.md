# <img src="https://github.com/vllm-project.png" width="24" style="vertical-align: middle; border-radius: 4px;" /> vLLM

High-throughput local LLM serving engine for model inference.

## Installation

```shell
pip install vllm
```

## Configuration

No basic configuration required.

## Start / Usage

```shell
vllm serve llama3
```
