# Terraform

Infrastructure as code tool for provisioning and managing cloud resources declaratively.

## Installation

```shell
brew install terraform
```

## Usage

```shell
# Initialize a working directory
terraform init

# Preview changes
terraform plan

# Apply changes
terraform apply

# Destroy resources
terraform destroy
```
