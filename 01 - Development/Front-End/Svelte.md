# <img src="https://github.com/sveltejs.png" width="24" style="vertical-align: middle; border-radius: 4px;" /> Svelte

A radical new approach to building user interfaces.

## Installation

```shell
npm create svelte@latest my-app
```

## Usage

```svelte
<script>
  let name = 'world';
</script>

<h1>Hello {name}!</h1>
```
