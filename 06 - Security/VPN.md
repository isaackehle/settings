# <img src="https://github.com/OpenVPN.png" width="24" style="vertical-align: middle; border-radius: 4px;" /> VPN

Virtual Private Network clients for secure remote access.

## Installation

```shell
# OpenVPN client
brew install --cask tunnelblick

# Proton VPN client
brew install --cask protonvpn
```

## Configuration

Import your VPN profile or sign in to your VPN provider account.

## Start / Usage

Start: Open the app from Applications.
